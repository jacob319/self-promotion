Learning the Basics of Html and CSS

My experience of learning Html and CSS has been an exciting journey that I cherish. Learning the structure Html and learning the styling in CSS has been a challenge at times but rewarding at others.

Html stands for Hyper Text Markup Language which describes the structure of web pages using markup. In Html you use different tags to render ones heading h1, h2, h3…h6, paragraph p. Most tags have closing tags

CSS stands for Cascading Style Sheets which describes how content will look on a page. CSS may display internal on html file or external as a .css file. With CSS one can change the color of text background. With CSS one could set margins and padding to make things neater.
